## Batter Code — руководство для администратора

### Доступ в настройки

1. Откройте приложение.
2. На главном экране введите **мастер‑пароль** (по умолчанию `000000`).
3. Нажмите `✓` — откроется экран **«Настройки»** с двумя пунктами:
   - «Мастер‑пароль»;
   - «Коды и фразы».

---

### 1. Мастер‑пароль

1. Нажмите пункт **«Мастер‑пароль»**.
2. Введите новый 6‑значный код (полная видимость, только цифры).
3. Нажмите **«Сохранить»**.  
   Новый мастер‑пароль вступит в силу.

Рекомендуется не использовать мастер‑пароль, совпадающий с пользовательскими кодами.

---

### 2. Управление кодами и фразами

#### 2.1. Экран «Коды и фразы»

Содержит список всех кодов:

- строка: `код` + статус (**активен/выкл**) + превью фразы;
- список может быть сколь угодно длинным.

#### 2.2. Добавить код

1. Нажмите **«+ Добавить код»**.
2. В форме:
   - поле **«Код (6 цифр)»** — только цифры, ровно 6 символов;
   - поле **«Фраза»** — любой текст, от 3 до 10 000 символов;
   - переключатель **«Код активен»**.
3. Заполните поля и нажмите **«Сохранить»**.

Если при сохранении флаг «Код активен» включён:

- все **другие записи с таким же кодом** автоматически переводятся в состояние **выкл**;
- всегда существует **не более одного активного** кода для одного и того же значения.

#### 2.3. Редактировать код

1. Нажмите по строке в списке.
2. Отредактируйте:
   - сам код (6 цифр),
   - текст фразы,
   - состояние **активен/выкл**.
3. Нажмите **«Сохранить»**.

Правило уникальности активного кода для одного значения сохраняется.

#### 2.4. Удалить код (свайп влево)

1. Проведите **влево** по нужной строке.
2. Подтвердите удаление в диалоге:
   - «Удалить код …?» → **«Удалить»** / **«Отмена»**.
3. При подтверждении код и его фраза удаляются окончательно.

#### 2.5. Активировать / деактивировать код (свайп вправо)

1. Проведите **вправо** по строке.
2. Код сразу переключит статус:
   - **активен → выкл** или **выкл → активен**.
3. При активации кодов с одинаковым значением:
   - текущий становится **активен**;
   - все остальные совпадающие коды становятся **выкл**.

---

### 3. Поведение на главном экране

- При вводе пользователем учитываются **только активные** коды.
- Деактивированные коды:
  - видны в настройках,
  - игнорируются при вводе на главном экране,
  - удобны для временного отключения сценариев без их удаления.
