## Batter Code — руководство для пользователя

### Что делает приложение

Приложение показывает экран ввода 6‑значного кода и анимацию в стиле «Матрицы» с финальной фразой для каждого кода.

### Как пользоваться

1. **Запуск**  
   Откройте приложение — вы увидите:
   - 6 пустых ячеек кода;
   - встроенную цифровую клавиатуру (0–9, `⌫`, `✓`).

2. **Ввод кода**
   - Нажимайте цифры — они по очереди появляются в ячейках.
   - Кнопка `⌫` стирает последнюю введённую цифру.
   - После ввода 6 цифр нажмите `✓`.

3. **Результат**
   - Если код **правильный и активный**:
     - сначала краткая «Проверка…»,
     - затем анимация «Матрица»,
     - затем появляется фраза, связанная с этим кодом.
     - нажмите крестик в правом верхнем углу, чтобы вернуться к вводу.
   - Если код **неверный**:
     - показывается загрузка и сообщение, что код неверный,
     - ячейки очищаются, можно ввести код снова.

4. **Примечания**
   - Количество кодов и тексты фраз задаёт администратор.
   - Обычный пользователь **не имеет доступа** к экрану настроек и не может менять коды/фразы.
